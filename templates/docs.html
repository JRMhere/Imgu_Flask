About this site

Welcome to ImguFlask! This web application, built with Flask, allows you to upload photos and apply filters to them.

The images are hosted on Imgur, using Imgur's open-use API - it was pretty simple to set up the API linkage, so take this site as a demonstration.

If I can do it, I have no doubt that you can as well.


How to use this site

Filters can be applied from the Library page or from the Index page - but you must upload an image first.

Keep in mind that at the end of each session, the image held in your index will be reset, so make sure to save the image to your library once you get one you really like.

Your user profile stores interesting statistics about the images you've uploaded, filtered, and saved to your library, so check out that tab if you're curious.
